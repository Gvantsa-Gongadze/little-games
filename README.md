# Little Games

A multiplayer mini-game platform.

## Stack
- Frontend: React + TypeScript + Vite
- 2D: Pixi.js v8
- 3D: Three.js r168
- Backend: Colyseus (Node.js)
- DB: Supabase
- Deploy: Vercel + Railway

## Setup

```bash
pnpm install
pnpm dev:client   # starts frontend on :5173
pnpm dev:server   # starts Colyseus on :2567
```
